Ключевые элементы пароекта

Используется сброс стилей reset.css

Шрифты получены с сайта https://fonts.google.com/

Логотип в виде svg-файла.
Иконки в footer оформлены в виде спрайта -- собраны в одном файле sprite.svg с помощью тега symbol

Для изображений используются свойства 
width: 100%; height: 100%; object-fit: cover
для того чтобы они могли пропорционально увеличиваться-уменьшаться под размер контента.

Все секции оформлены с помошью контейнера:
<header>
  <div class="container">
    ...
  </div>
</header>

С помошью класса container задается:
1) Автоматические margins слева-справа для выравнивания контента по центру;
2) Маленькие padding слева-справа, чтобы контент не прилипал к границам окна браузера при уменьшении ширины окна браузера
3) max-width: 1170px;
C помощью класса-родителя container (header, footer, section) задется заливка фона секции и padding для секции сверху-снизу.

Для верстки контента используются flex-box модель, с помощью установки на родителя элементов display: flex:
1) Логотип в меню слева, а группа меню справа -- с помощью justify-content: space-between
2) Расстояние между элементами меню, между колонками в секции Feature, между иконками в footer -- с помощью gap
3) Секции About (etc) и Feature бьются на две-три колонки с помощью свойства flex, например:
   flex: 1 1 50% (flex-grow, flex-shrink, flex-basis)
4) Блоки в секции Feature имееют одинаковую высоту за счет свойства по умолчанию align-items: stretch;
5) Ссылки в секции Feature находятся в конце блока -- с помощью flex-direction: column, а в самих ссылках margin-top: auto;
